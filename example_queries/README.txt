
Example queries for CLEF e-Health 2015
Task 2: User-Centred Health Information Retrieval

Last update: 5 Mar 2015
===============================================================================

Files:
------

* clef2015.training.queries-EN.txt
-------------------------------------
This file contains 5 example queries (in English). 
This year we will explore circumlocutory queries that users may pose when 
faced with signs and symptoms of a medical condition. 
To form the query set, we adopted the method recently 
investigated by [1,2].

Every query was translated from English to a target language (AR: Arabic, 
CS: Czech, DE: German, ES: Spanish, FA: Farsi, FR: French, IT: Italian, 
PT: Portuguese).

We also provide a baseline translation made by Google Translator (files containing
"auto" in their names).
-------------------------------------

* translations.xml
-------------------------------------
We provide the translations.xml file for an easier visualization 
of the automatically generated translations.
-------------------------------------

* clef2015.qrels.training.txt
-------------------------------------
A pool containing the top 10 results of different systems was made and
manually evaluated. Only the query field of each topic was used.
-------------------------------------

* clef2015.qrels.training.additional.txt
-------------------------------------
The same procedure described above was used to generate this file, but
the disease name from <narr> field was manually appended to the query.
